Help

We're sorry that you're having issues with Adventurer's Codex, but we would appreciate it if let us know what's going wrong. If you have any issues, questions, or concerns let us know by shooting us an email.

Feedback

We here at Adventurer's Codex are committed to creating the best tabletop RPG app we can, and we'd love to hear from you with any advice, comments, questions, or feature requests. We can't promise to implement them all, but we do read all feedback that comes in. Please let us know if we can improve the app in any way, or if you'd want to help us and implement new awesome features.

For Users

If you have any questions, comments, feedback, or concerns please send us an email.

For Developers

If you'd like to contribute to Adventurer's Codex, then file an issue on GitHub, or shoot us an email to get access to the developer chat room!
